# 进度成就

当玩家完成一定数量成就时，便会触发此种类型成就。

- **名称**：`进度成就`
- **类型名**：`achiCount`
- **json**：

```json
{
    "achiCount": {
    "enable": true,
    "name": "成就数量成就",
    "details": {},
    "regx": {}
  }
}
```



## 小有名气

- 触发值：`10`
- 词条：`达成10个成就`
- 条件：`达成10个成就`
- json：

```json
{
    "10": {
        "enable": true,
        "msg": "小有名气",
        "condition": "达成10个成就"
      }
}
```
