# 物品成就

玩家获得物品时会触发此成就。

- **名称**：`物品成就`
- **类型名**：`itemObtain`
- **默认开启**：`true`
- **json**：

```json
{
	"itemObtain": {
		"enable": true,
		"name": "物品成就",
		"details": {},
		"regx": {}
	}
}
```




## 聊的火热!

- 触发值：`minecraft:furnace`
- 词条：`聊的火热!`
- 条件：`首次获得熔炉`
- 默认开启：`true`
- json：

```json
{
	"minecraft:furnace": {
		"enable": true,
		"msg": "聊的火热!",
		"condition": "首次获得熔炉"
	}
}
```
