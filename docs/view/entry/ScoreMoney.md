# 经济成就

成就描述

- **名称**：`经济成就`
- **类型名**：`ScoreMoney`
- **默认开启**：`true`
- **json**：

```json
{
	"ScoreMoney": {
		"enable": true,
		"name": "经济成就",
		"details": {},
		"regx": {}
	}
}
```




## 大负翁

- 触发值：`${}&#60;=0`
- 词条：`大负翁`
- 条件：`经济小于等于0`
- 默认开启：`true`
- json：

```json
{
	"${}<=0": {
		"enable": true,
		"msg": "大负翁",
		"condition": "经济小于等于0"
	}
}
```
