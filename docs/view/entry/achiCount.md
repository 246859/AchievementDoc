# 进度成就

也有人曾经建议希望增加一个完成所有成就才能达成的成就，不过此成就插件的词条数量，类型未来可能会变化的十分频繁，所以很难去准确的判断是否完成所有成就，所以全成就这个想法不予考虑。

- **名称**：`进度成就`
- **类型名**：`achiCount`
- **默认开启**：`true`
- **传入值**：`玩家成就完成的数量`
- **词条数量**：`5`
- **json**：

```json
{
	"achiCount": {
		"enable": true,
		"name": "进度成就",
		"details": {},
		"regx": {}
	}
}
```
