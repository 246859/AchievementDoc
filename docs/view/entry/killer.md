# 击杀成就

玩家击杀生物时会触发此成就

- **名称**：`击杀成就`
- **类型名**：`killer`
- **默认开启**：`true`
- **传入值**：`生物的标准类型名`
- **词条数量**：`35`
- **json**：

```json
{
	"killer": {
		"enable": true,
		"name": "击杀成就",
		"details": {},
		"regx": {}
	}
}
```
