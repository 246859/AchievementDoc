# 射击成就

玩家用弓箭在一定距离内击中生物便会触发此类成就。

- **名称：`射击成就`**
- **类型名：`shootDistance`**
- **json：**

```json
{
	"shootDistance": {
    "enable": true,
    "name": "射击成就",
    "details": {},
    "regx": {}
  }
}
```





## 十米开外

- 触发值：`20`
- 词条：`十米开外`
- 条件：`用箭命中距离20以外的生物`
- json：

```json
{
	"20": {
        "enable": true,
        "msg": "十米开外",
        "condition": "用箭命中距离20以外的生物"
      }
}
```
